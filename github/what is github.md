# how upload local data on github via gitbash

  1. download git software 
  2. https://git-scm.com/install/windows
  3. open your main folder to upload data on github
  4. all commands to upload data on github 

    ```
    git init
    git add .
    git commit -m "first commit"
    git remote add origin https://github.com/Brijesh1990/ds-da-mwf-3pm.git
    git push -u origin main

    ```
